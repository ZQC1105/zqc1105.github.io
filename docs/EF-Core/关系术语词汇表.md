### 关系术语词汇表（MD 版）

| 术语 | 定义 |
|---|---|
| **依赖实体** | 包含外键属性的实体，又称“子级”。 |
| **主体实体** | 包含主键/备选键的实体，又称“父级”。 |
| **主体键** | 唯一标识主体实体的属性（主键或备选键）。 |
| **外键** | 依赖实体中存储主体键值的属性。 |
| **导航** | 实体上指向相关对象的属性。 |
| **集合导航** | 指向多个相关对象的导航（一对多/多对多中的“多”）。 |
| **引用导航** | 指向单个相关对象的导航（一对一/一对多中的“一”）。 |
| **反向导航** | 关系另一端对应的导航属性。 |
| **自引用关系** | 依赖实体与主体实体为同一类型。 |
| **必需关系** | 外键不可为空，依赖实体必须存在对应主体。 |
| **可选关系** | 外键可为空，依赖实体可独立存在。 |
| **双向关系** | 两端均定义了导航属性。 |
| **单向关系** | 仅一端定义了导航属性。 |
